Temperature Converter

A simple web-based temperature converter built with HTML, CSS, and JavaScript.
It allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin.

Features

Convert temperature between any two units: Celsius, Fahrenheit, Kelvin

Input validation: Alerts if the user enters an invalid number

Responsive and clean design with a light blue background

Instant conversion on button click

Rounded output to 2 decimal places

Technologies Used
| Technology     | Purpose                                        |
| -------------- | ---------------------------------------------- |
| HTML5          | Structure of the webpage                       |
| CSS3           | Styling and layout                             |
| JavaScript     | Temperature conversion logic and interactivity |
